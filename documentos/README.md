# Documentos oficiales

Use esta carpeta para cargar documentos PDF aprobados por la entidad, por ejemplo:

- Estatutos o acta de constitución.
- Certificación de cumplimiento de requisitos.
- Estados financieros o estado financiero de apertura.
- Relación de fundadores, directivos y cargos de control, cuando aplique.
- Certificaciones y soportes exigidos para el trámite correspondiente.

Después de cargar los PDF, agregue los enlaces en la sección `Registro Web DIAN` del archivo `index.html`.
